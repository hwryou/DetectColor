Program to detect specific colors. Written in Python, by utilizing OpenCV and numpy.

This program will detect the color by detecting rectangle in specific color. It may not work
if the object with specific color is not big enough or have inappropriate shape.

To add more colors to detect or edit the colors to detect, please refer the guidelines
written in program.
